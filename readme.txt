Moko is an N9 app explorer to explore,try and share newest apps for N9.
Moko depends on the following components to be present in the save folder level.  
Feedparrot for feed management. Clone in from https://sriks@github.com/sriks/feedparrot.git and
Shareui for presenting Harmattan share ui. Clone it from https://sriks@github.com/sriks/shareui.git
Happy coding.
